# shimo-auto-download

## 项目说明

本项目为解决自动下载石墨文档上的文件到本地存档



## 运行环境

+ Python 3.8
+ requests 库版本 2.25.1
+ 可使用 pip install -r requirements.txt 进行安装 requests



## 项目介绍

+ 项目结构

  ```
  shimo-auto-download
  │  .gitignore
  │  config.py
  │  README.md
  │  requirements.txt
│  run.py
  ```

  config.py : 项目的配置文件,使用前需要在此进行配置

  run.py：进行文件下载的文件

  requirements.txt ：安装的库的版本文件
