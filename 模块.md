# 模块

1、包

包目录下必须有 __init__.py 文件，否则该目录会被识别为普通目录

2、作用域

变量或函数名以_开头，是可以被隐藏起来的，不能直接被直接引用

```
def _private_1(name):
    return 'Hello, %s' % name

def _private_2(name):
    return 'Hi, %s' % name

def greeting(name):
    if len(name) > 3:
        return _private_1(name)
    else:
        return _private_2(name)
```
